# QA Test Project

Учебный проект по тестированию интернет-магазина

## Описание
В рамках проекта проводилось тестирование основного функционала веб-приложения.

## Что протестировано
- Поиск товаров
- Фильтрация товаров
- Добавление в корзину
- Регистрация пользователя

## Используемые инструменты
- Ручное тестирование
- Postman (API тестирование)
- SQL (базовый уровень)
- Python + Selenium (автотесты)

## Структура проекта
- test-cases — тест-кейсы
- checklists — чек-листы
- bug-reports — баг-репорты
- postman — API тесты
- autotests — автотесты
